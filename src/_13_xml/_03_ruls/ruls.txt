1. xml документ объявляет строка <?xml version="1.0" encoding="UTF-8" ?>
    в строке определяетс яверсия 1.0 и кодировка
2. xml документ должени иметь один и только один корневой элемент - root
    все остальные элементы раположены внутри корневого
    вне корневого элемента нельзя размещать другие элементы
3. каждый элемент определяется с помощью открывающегося и закрывающегося тэга <> </>
    также существует сокращенное объявление внутри которого можно размещать данные
    <cat id="3" name="Mashka" age="7"/>
    данный тэг должен быть закрыт сразу
4. элемент может иметь вложенные элементы и аттрибуты
5. Аттрибуты определяются в теле элемента и меют вид название="значение"
6. внутри простых элементов размещается их значение
    <name>Barsik</name>
7. развания элементов регистрозависимые
    <name> и <Name> - это разные тэги
